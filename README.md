# Rock Paper Scissors Game 

A simple Rock Paper Scissors game built using HTML and JavaScript.  
This is a beginner-friendly project to practice JavaScript basics like:
* Random number generation
* Conditional statements
* Event handling



# Features
* Play Rock, Paper, or Scissors against the computer.
* Computer makes a random move every time.
* Displays result in **alert** and logs it in the **console**.



# Technologies Used
* HTML
* JavaScript
